Для установки программы нужно выполнить пару действий
1 закинуть файлы movie2.mdf, movie2_log.ldf на диск С
2 закинуть папку 54 на диск С
3 запустить файл setup.exe в папке 44 и дождаться завершения процеса
4 после установки приложение запуститься автоматически, в дальнейшом заходить в приложение через меню пуск 
(для входа в вкладку admin(admin| 1234) для входа в вкладку staff(test| 1234)

Для встановлення програми потрібно виконати пару дій

2 закинути папку 54 на диск C
3 запустити файл setup.exe у папці 44 і дочекатися завершення процесу
4 після встановлення програма запуститися автоматично, надалі заходити в програму через меню пуск 
(для входу до вкладки admin(admin| 1234) для входу до вкладки staff(test| 1234)

To install the program, you need to perform a couple of steps
1 throw the files movie2.mdf, movie2_log.ldf on the disk
2 throw the folder 54 on the disk
3 run the setup.exe file in folder 44 and wait for the process to complete
4 after installation, the program will start automatically, then enter the program through the start menu 
(to enter the admin tab(admin| 1234) to enter the staff tab(test| 1234)
